# XPlanar Application Library
This library is built on the base functionality provided by TF5890 and its main goal to provide a consistent and flexible jumping off point for using XPlanar in applications.  

## Note: Version 2.0 is not a direct compatability upgrade. See release notes in repository or wiki for potential breaking changes!
https://github.com/Beckhoff-USA-Community/XPlanarApplicationLibrary/wiki/Version-2.0-release-notes


## See below for version compatability
## ***********************************************************
**TF5890 v4.1.x > XPlanarAppilcation v1.0 > Tc4024 Only

**TF5890 v4.2.x > XPlanarAppilcation v1.2 > Tc4024 / Tc4026

**TF5890 v4.4.x > XPlanarAppilcation v1.4 > Tc4026 Only

**TF5890 v5.0.x > XPlanarAppilcation v2.x > Tc4026 Only

## Requirements

The following components must be installed via TwinCAT Package Manager to use this library:

- TE1000 TwinCAT 3 Engineering
- TF5890 TwinCAT 3 XPlanar
- TF5400 TwinCAT 3 Advanced Motion Pack
